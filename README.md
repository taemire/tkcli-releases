# TKCLI Releases

TKCLI(TACHYON CLI) 바이너리 배포 저장소입니다.

## 다운로드

[Releases 페이지](https://github.com/taemire/tkcli-releases/releases)에서 최신 버전을 다운로드하세요.

## 포함 파일

| 파일 | 설명 |
|------|------|
| `tkcli` | Linux amd64 바이너리 |
| `tkcli_*.zip` | 릴리즈 아카이브 |
| `TKCLI_USER_MANUAL_*.pdf` | 사용자 매뉴얼 |

## 설치

```bash
# 다운로드 후 실행 권한 부여
chmod +x tkcli
sudo mv tkcli /usr/local/bin/

# 버전 확인
tkcli version
```
